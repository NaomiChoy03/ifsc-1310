Thoughts on Progressive Enhancement

Progressive Enhancement is a strategy that allows users to access the basic content of a website. Users with better Internet access or additional brower features can use a more enhance version of the website. Progressive enhancement is important in modern web development because it is a good way for web developers to balance issues relating to building websites that can be accessed by people using unknown browsers.
